# direct-delete-sql-maker

## 環境

- MacOS Big Sur
- Python 3.7.12
- MySQL 5.6

## Usage

```sh
python src/make_delete_company.py {company_id}
```

## setup

- clone
  ```sh
  git clone https://github.com/kim-yuchul-okan/direct-delete-sql-maker.git
  ```
- library install
  ```sh
  pip install mysqlclient
  ```
